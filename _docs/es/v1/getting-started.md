# Primeros pasos con el Portal de Documentación

Para empezar:

1. Elige tu idioma en el selector de idioma.
2. Explora las secciones de documentación: Introducción, Primeros pasos, Referencia de API.
3. Copia ejemplos de código directamente desde el portal.
4. Proporciona comentarios usando el widget de retroalimentación.
