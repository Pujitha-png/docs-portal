# Pour commencer avec le Portail de Documentation

Pour commencer :

1. Choisissez votre langue dans le sélecteur de langue.
2. Explorez les sections de documentation : Introduction, Pour commencer, Référence API.
3. Copiez les exemples de code directement depuis le portail.
4. Donnez votre avis via le widget de feedback.
