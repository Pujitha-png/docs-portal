# Erste Schritte mit dem Dokumentationsportal

So starten Sie:

1. Wählen Sie Ihre Sprache im Sprachumschalter.
2. Erkunden Sie die Dokumentationsabschnitte: Einführung, Erste Schritte, API-Referenz.
3. Kopieren Sie Codebeispiele direkt aus dem Portal.
4. Geben Sie Feedback über das Feedback-Widget.
